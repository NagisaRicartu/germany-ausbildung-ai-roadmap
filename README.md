* `evidence/`：必要的运行证据和截图

## 学习原则

1. 先独立思考和尝试，再使用 AI。
2. AI 可以解释、检查和提出建议，但我必须理解最终提交的内容。
3. 每个 commit 只完成一个清晰的任务。
4. commit 前检查改动，不提交密码、API Key 或个人敏感资料。
5. 每周进行一次闭卷重建和学习复盘。
